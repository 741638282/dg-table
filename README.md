# dg-table

#### 项目介绍
基于element ui table二次开发的多功能实现远程筛选的表格
<br>
<br>
提供的筛选器：
  - 从服务器拉取匹配项列表
  - 单纯的文本搜索
  - 提供单选的列表
  - 联级选择器（可用于地区搜索）
  - 日期筛选
  - 范围筛选

<br>
提供分页，多选，自定义列等配置项
